Tailles du texte courant


Pour attribuer une taille au texte, vous devez utiliser l'attribut size puis spécifier la taille dont les limites vont de 6pts à 20pts.



<p class="size-12"> Texte taille 12 </p>

Citations


Les citations sont des éléments HTML plutôt optionnels mais ils peuvent consistuer un bon style visuel pour les citations dans des articles ou autres. Dans l'exemple ci-dessous, en plus de la classe blockquote-couleur, nous avons ajouté d'autres classes pour styliser le texte. Vous êtes bien sûr libres de styliser ce dernier comme bon vous semble.
